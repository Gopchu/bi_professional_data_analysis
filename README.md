# Data Professional Survey Breakdown
## Project Objective
In this project, we are going to take a look at the results of our survey, which contains information and data regarding our employees, their salaries and overall satisfaction with their current arrangments

## Dataset Used
- <a href="https://github.com/Gopchu/bi_professional_data_analysis/blob/main/Power%20BI%20-%20Final%20Project.xlsx">Survey Data<a/>

## Questions/Analysis

-Country of origin  
-Average salary  
-Average salary distribution based on sex  
-Work/life happiness  
-Salary happiness  
-Favorite programming language, a question that could imply certain future career trends  

## Process

-First things first, we had to make sure that the data is clean and ready for analysis.  
-In order to do so, we imported .xlsx file and using "Transfor Data" option in Power BI, we have completely bypassed need to use Microsoft Excel.  
-Next, in order to accomplish our analysis, we need to decide which of the data is necessary to focus on, removing uneccessary columns, cleaning and sorting the important ones, and making sure that it is a correct data type as well as creating additional column, in this case, "average salary".
